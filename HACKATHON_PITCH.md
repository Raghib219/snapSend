# 🎯 SnapSpend - Real-Time Budget Nudge Agent
## National Level Hackathon Pitch

---

## 🚨 The Problem

**Indians are overspending on food delivery apps!**

- Average person spends ₹8,000-12,000/month on Swiggy/Zomato
- 73% don't track their spending
- Traditional finance apps are boring and ignored
- No real-time intervention when overspending happens

**Result:** Financial stress, depleted savings, poor money habits

---

## 💡 Our Solution

**SnapSpend: Your AI Financial Guardian with Personality**

A real-time budget nudge agent that:
- ✅ Monitors spending patterns automatically
- ✅ Sends funny, personalized nudges when overspending
- ✅ Predicts future overspending before it happens
- ✅ Suggests smart alternatives to save money
- ✅ Gamifies saving with achievements and rewards

## 💻 Technical Stack

### Frontend
- React + TypeScript
- Tailwind CSS + shadcn/ui
- Chart.js for visualizations
- Modern, responsive design

### Backend
- Node.js + Express
- Google Gemini AI
- RESTful APIs
- CSV processing

### Key APIs
```
POST /analyze-transactions      → Upload & analyze
POST /generate-nudges          → Real-time alerts
POST /predict-overspending     → Predictive warnings
POST /suggest-alternatives     → Smart savings
GET  /check-achievements       → Gamification
POST /ask-question            → AI chatbot
```
